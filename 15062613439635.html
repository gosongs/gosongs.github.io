设置mysql密码报错

报错日志：

error: 'Access denied for user 'root'@'localhost' (using password: NO)'

解决办法：

先结束mysql进程。

# 安全模式启动
/usr/bin/mysqld_safe --skip-grant-tables >/dev/null 2>&1 &

# 进入mysql
/usr/bin/mysql -u root mysql

# 修改密码
update user set password = Password('newpassword') where User = 'root';
