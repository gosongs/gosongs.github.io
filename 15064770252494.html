初始化 Angular4 项目

使用 angular-cli

# 全局安装 angular-cli
npm install -g @angular/cli

# 初始化项目
ng new ng4demo

cd ng4demo

# 安装依赖
yarn install

# 启动项目
yarn start

angular-cli 可选指令：

  • ng new：ng new [name] 创建一个 ng 新项目。
  • ng serve
  • ng generate
  • ng lint
  • ng test
  • ng e2e
  • ng build
  • ng get/ng set
  • ng doc
  • ng eject
  • ng xi18n

整合 material2

官方教程。

第一步：安装依赖

@angular/material @angular/cdk

第二步：引入图标和主题

在全局样式 style.css 中引入图标（可选）和material主题样式（必选）：

/* style.css */
@import '~https://fonts.googleapis.com/icon?family=Material+Icons';
@import '~@angular/material/prebuilt-themes/deeppurple-amber.css';

第三步：手势支持（可选）

yarn add hammerjs

在项目入口 main.ts 中引入 hammerjs：

import 'hammerjs'

第四步：将material的模块注入到项目模块中

修改 app.modules.ts：
```javascript
// app.modules.ts
import { BrowserModule } from '@angular/platform-browser';
import { NgModule } from '@angular/core';
import { MdButtonModule } from '@angular/material'; // 引入button模块

import { AppComponent } from './app.component';

@NgModule({
declarations: [
AppComponent
],
imports: [
BrowserModule,
MdButtonModule // 将button模块注入到app模块中
],
providers: [],
bootstrap: [AppComponent]
})
export class AppModule { }
```

在 app.component.html 中使用上边注入的 button 模块：

<button md-button>Click me!</button>
<button md-button>Click me!</button>
<button md-button>Click me!</button>
<button md-button>Click me!</button>
<button md-button>Click me!</button>
