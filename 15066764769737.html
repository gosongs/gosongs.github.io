常用 JS 库

通知类

  • notice: A clean and simple notification, input, and selection suite for javascript, with no dependencies.
  • iNotify: JS 实现浏览器的 title 闪烁、滚动、声音提示、chrome、Firefox、Safari等系统通知。

时间处理类

  • moment.js
  • spacetime.js, 比 momentjs 轻量.

设备检测类

  • device.js: Device.js makes it easy to write conditional CSS and/or JavaScript based on device operating system (iOS, Android, Blackberry, Windows, Firefox OS, MeeGo), orientation (Portrait vs. Landscape), and type (Tablet vs. Mobile).

存储类

  • cookie.js: A tiny JavaScript library that simplifies cookies.
  • store.js: Cross-browser storage for all use cases, used across the web.

CSS 动画

  • move.js: Move.js is a small JavaScript library making CSS3 backed animation extremely simple and elegant.

数学运算

  • math.js: An extensive math library for JavaScript and Node.js

分享

  • share.js: 一键分享到微博、QQ空间、QQ好友、微信、腾讯微博、豆瓣、Facebook、Twitter、Linkedin、Google+、点点等.

工具库

  • lodash: A modern JavaScript utility library delivering modularity, performance, & extras.
  • is.js: Check types, regexps, presence, time and more...

其他

  • favico.js: Make use of your favicon with badges, images or videos.
  • jieba.js: 文本内容分词和关键词抽取.
